Workspace menu — layout options

One merged control on the header right · carries session + checkout · "detached" when the branch is in no worktree · limn

What this control is: a single chip that replaces the separate Session and Edit-target chips. Its label reads out both axes — the current review (Session #4 or a draft Session ‹new›) and where the compare branch lives (primary, or detached when it's checked out nowhere). Opening it lets you switch reviews and switch / set the checkout. Detached ⇒ read-only (agent locked); you can still browse and save comments.

1Stacked — Reviews over CheckoutRecommended

Status line up top, then two labelled lists in one scroll: Reviews on this branch (switch between them; the draft is a row, New review adds one), then Checkout (the worktree the branch lives in; detached ⇒ pick one + a checkout button). Reads top-to-bottom; both axes visible at once.

Why this oneBoth axes always visible — you see the review and the checkout state in one glance. Natural reading order. The checkout button has a stable home at the bottom.
Watch-outCan get tall with many sessions + worktrees. Cap each list (e.g. 4) with a filter/“+N more”.
2Two columns — Reviews | Checkout

The two axes side by side, making their independence explicit: pick a review on the left, a worktree on the right. The checkout action spans the bottom. Less vertical scroll; wider popover.

StrengthVisually states "these are two separate things." No scrolling between axes. Good if both lists are short.
Watch-outWide (~520px) for a header dropdown. Two scrolling columns get awkward if either list is long. Heavier than the common "just switch review" need.
3Tabs — Reviews ⇄ Checkout

A segmented toggle; one pane at a time. The inactive tab still flags state (Checkout · detached), so you know to look. Compact width, but you only see one axis until you switch.

4Status-led — one CTA, compact switchers

Lead with the state and the single most useful action (when detached: Check out to edit, a split button that picks the worktree). Switching review or worktree is secondary, as compact pill rows below. Optimized for the common path: "I'm detached → get me working."
